# 🎮 Java Number Guessing Game

A number guessing game made in Java for learning purposes.

This project was created while studying Java and practicing basic programming concepts.

## Features

- Number guessing game
- Limited number of attempts
- Hints for "too high" or "too low"
- Displays attempts used
- Reveals the secret number if the player loses

## Technologies

- Java (JDK 21)

## How to Run

1. Clone the repository

git clone https://github.com/nekotsumidev/java-number-guessing-game.git

2. Open the project in your Java IDE

3. Run the main class
